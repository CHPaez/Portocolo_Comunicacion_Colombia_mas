Ejemplo – Flujo por Procesos

Flujo – Proceso de Contratación

Paso 1: Registrar Solicitud

El líder debe ingresar los datos del cargo solicitado.

Paso 2: Validación por Gestión Humana

El área revisa disponibilidad y autorizaciones.

Paso 3: Entrevista y Selección

Se agenda entrevista, pruebas y elección del candidato.

Flujo – Solicitud de Capacitación

Paso 1: Diligenciar Formato

El colaborador debe llenar el formato oficial.

Paso 2: Envío y Revisión

Enviar el formato y esperar validación.

Paso 3: Confirmación

Gestión Humana confirma aprobación o fechas.

Flujo – Trámites Administrativos

Paso 1: Seleccionar Trámite

Certificados, novedades, desprendibles, etc.

Paso 2: Enviar Documentos

Adjuntar soporte según el trámite solicitado.

Paso 3: Confirmación de Proceso

Gestión Humana confirma el avance o finalización.
